## Descripción

Este es un portafolio web personal que presenta el trabajo y habilidades de **Carlos Romero**, un desarrollador web apasionado por crear experiencias digitales limpias, funcionales y accesibles.

La página sirve como vitrina profesional donde se muestran proyectos destacados, información personal y habilidades técnicas. El diseño enfatiza la **modernidad, profesionalismo y sencillez**, permitiendo a visitantes conocer mejor el trabajo del desarrollador y establecer contacto.

## Tecnologías Utilizadas

### Frontend

- **HTML5**: Estructura semántica y accesible
- **CSS3**: Estilos modernos con:
  - Variables CSS personalizadas
  - Gradientes radiales
  - Flexbox y Grid Layout
  - Diseño responsivo (Mobile First)
  - Backdrop filter para efectos visuales
  - Animaciones suave (smooth scroll)

### Características

- **Sin dependencias externas**: Desarrollado con HTML y CSS puros (sin frameworks)
- **Responsivo**: Diseño adaptable a dispositivos móviles y desktop
- **Contenido editable**: Fácil de mantener y actualizar
- **Tema oscuro moderno**: Esquema de colores profesional con azul y gris
